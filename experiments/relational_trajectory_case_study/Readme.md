long-term dialogue with a persistent AI system).

→ model-transition discontinuity → recovery) detected using the OSD scoring
pipeline, along with a corrected methodological analysis of inter-judge
agreement (see analysis_draft.md §5 for an example of a conclusion revised
after full-dataset recomputation).
